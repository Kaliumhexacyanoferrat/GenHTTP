100% Managed

Purely written in C# and provided as a .NET Standard 2.0 library, with no further dependencies.

Open Source

All sources are available on GitHub under the MIT license.

Flexible Hosting

Embed the engine into your .NET application or host it via Docker.

Features

Originally intended to run on a netbook with an Intel Atom processor, the server code is optimized to be fast and minimalistic. For example, this website is served by a Raspberry Pi.

The server engine is intended bring along as few dependencies as possible. The process won't fuzz around with your file system, threads, statics, or assemblies.

While being minimalistic, the server is designed to write simple web apps in a short time. Therefore, it features optional modules to handle resources, add webservices, or use templates.

To get started, check out the samples on GitHub.
