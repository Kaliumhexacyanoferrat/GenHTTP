# GenHTTP.Modules.ClientCaching

This package provides two concerns that allow
to instruct the client to cache the responses
generated by the server. Please refer to the
documentation for samples and additional
details:

- [Policy](https://genhttp.org/documentation/content/concerns/client-caching-policy/)
- [Validation](https://genhttp.org/documentation/content/concerns/client-caching-validation/)
